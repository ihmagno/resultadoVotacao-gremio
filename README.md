## Dashboard de Votação do Grêmio Estudantil do Colégio Adventista
Este projeto apresenta uma solução para visualizar e analisar os resultados da votação para o Grêmio Estudantil do Colégio Adventista. A votação foi realizada utilizando o Google Forms, e os dados brutos foram processados e transformados em uma dashboard interativa.

### Recursos Principais
Integração com Google Sheets: Conexão direta com a API do Google Sheets para acessar os dados da votação em tempo real ou após a conclusão do pleito;
<br>
Processamento de Dados com Python: Utilização da biblioteca Pandas para a criação de um DataFrame a partir dos dados brutos, facilitando a manipulação e a preparação para a visualização;
<br>
Visualização Clara e Objetiva: Os resultados da votação são exibidos através de um gráfico de pizza, proporcionando uma visão imediata e percentual da distribuição dos votos entre os candidatos ou chapas.
### Tecnologias Utilizadas
Python: Linguagem principal para a lógica e processamento de dados.
<br>
Pandas: Essencial para a estruturação e limpeza dos dados do Google Sheets em um formato utilizável (DataFrame).
<br>
Google Sheets API: Para a extração dos dados da planilha de resultados da votação.
<br>
Matplotlib/Seaborn/Plotly (Sugestão, caso tenha usado uma destas): Biblioteca de visualização para gerar o gráfico de pizza.
